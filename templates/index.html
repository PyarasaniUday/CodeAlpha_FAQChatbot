AI FAQ Chatbot

Ask beginner-friendly Artificial Intelligence and Machine Learning questions. This chatbot is specially designed for the AI/ML domain and searches a curated FAQ knowledge base to intelligently find and return the most relevant answer using NLP, TF-IDF vectorization, and cosine similarity.

System Pipeline

🎤
Input signal received

You can type or use voice input for AI, ML, NLP, CNN, SVM, KNN, and project questions.

🧠
NLTK preprocesses text

The backend converts text to lowercase, tokenizes it, and removes common stopwords.

⚡
TF-IDF maps intent

Each FAQ question and user query becomes a numerical vector using scikit-learn.

🎯
Cosine similarity resolves answer

The closest FAQ match is returned with a confidence score and spoken back to the user.
